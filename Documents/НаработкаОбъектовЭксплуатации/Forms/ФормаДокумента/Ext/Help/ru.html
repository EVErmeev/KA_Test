Наработка объектов эксплуатации

Документ предназначен для отражения текущих значений показателей эксплуатации, определяющих натуральное выражение степени износа объекта эксплуатации.

Для правильного заполнения документа в строках таблицы на закладке Наработки должны быть указаны:

  • Объект эксплуатации - объект, для которого регистрируется текущее значение наработки;
  • Показатель наработки - показатель текущего значения наработки;
  • Значение - текущее значение наработки.
