Правила получения фактических данных по статьям бюджетов

Правила предназначены для настройки получения фактических данных по Статьям бюджетов.
Сбор фактических данных производится либо в бюджетных отчетах, предполагающих вывод фактических данных, либо при планировании, при заполнении планов на основе фактических данных, в документеЭкземпляра бюджета.

Правила получения фактических данных бюджетирования настраиваются для бюджетных статей и показателей. Для статей бюджетов правила могут быть настроены как для собственно получения фактических оборотов по статье, в этом случае указывается предопределенный сценарий Фактические данные, так и для указания правил для контроля лимитов расходования ДС, для этого применяется сценарий Исполнение бюджета.


Дополнительный отбор позволяет ограничить получаемые данные источника.

Существуют два способа определения аналитик источника данных:

  • При включенном режиме Расширенной настройки заполнения аналитик выбирается способ и значение заполнения каждой аналитики при получении фактических данных.
  • При отключенном режиме Расширенной настройки заполнение аналитик определяется системой автоматически, на основании типов данных имеющихся в источнике.
  • Принудительно по всем документам, которые нужно отразить в бюджетировании, с помощью команды Отразить документы в рабочем месте Настройка правил получения фактических данных
  • Автоматически по расписанию, заданном в рабочем месте Настройка правил получения фактических данных
  • Автоматически при формировании бюджетных отчетов за период отчета, если установлен флаг Автоматически запускать фоновое задание расчета хранимых данных бюджетирования из отчетов в разделе НСИ и администрирование - Бюджетирование и планирование - Бюджетирование
  • Автоматически при проведении документов, если установлен флаг Автоматически запускать фоновое задание расчета хранимых данных при проведении документов в разделе НСИ и администрирование - Бюджетирование и планирование - Бюджетирование
